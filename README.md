 ### Prerequisites
 - Python 3.10.2
 - OpenCV
 - easygui
 - numpy
 - matplotlib.pyplot
 - tkinter
 
 
 ### Getting Started
 
  Installing the necessary modules
  - For OpenCV
  
  ```html
pip install opencv-python
```

  - For easygui
  
  ```html
pip install easygui
```
  - For numpy
  
  ```html
pip install numpy
```
  - For matplotlib.pyplot
  
  ```html
pip install matplotlib
```
  - For tkinter
  
  ```html
pip install tk
```
